TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Port 5000 in use?
→ Kill it: lsof -i :5000 | grep LISTEN | awk '{print $2}' | xargs kill -9

Virtual environment issue?
→ Recreate: rm -rf venv && python3 -m venv venv

Module not found?
→ Install: source venv/bin/activate && pip install -r requirements.txt
